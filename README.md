# ✈️ Air Message

**A Creative Air Banner Messaging Experience Built with HTML, CSS & JavaScript**

✈️ Flying airplanes • 💬 Custom banner messages • 🎨 Retro pixel-art style • ⚡ Interactive animations

Air Message is a fun and interactive frontend project that allows users to create custom flying banner messages. Pixel-art airplanes carry personalized text across the screen, creating a nostalgic sky advertising experience inspired by retro games and classic air banners.

Designed for learning **JavaScript animations, DOM manipulation, dynamic content generation, and creative frontend development**.

---

# ✨ Features

✈️ **Animated Airplanes**

Pixel-art aircraft fly smoothly across the screen.

💬 **Custom Banner Messages**

Create and display personalized flying messages.

🎨 **Retro Pixel-Art Design**

Classic arcade-inspired visuals and styling.

⚡ **Dynamic Content Generation**

Messages are generated instantly from user input.

🌈 **Random Aircraft Colors**

Each airplane uses unique color combinations.

📱 **Responsive Layout**

Works across desktop, tablet, and mobile browsers.
